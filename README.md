Painter
=======

The Painter project is a simple web application that allows users to draw and paint on a canvas using different colors and brush sizes. The project is built using HTML, CSS, and JavaScript.

Features
--------

* A user-friendly interface for drawing and painting on a canvas.
* A color picker for selecting different colors.
* A brush size slider for adjusting the size of the brush.
* A "Clear Canvas" button to clear the canvas.
* A "Save Image" button to save the image to the user's device.

Directory Structure
-------------------

* `index.html`: This file contains the HTML code for the application.
* `styles.css`: This file contains the CSS styles for the application.
* `script.js`: This file contains the JavaScript code for the application.

To use the Painter project, simply clone the repository and open the `index.html` file in a web browser. The application will be displayed.

Overall, the Painter project is a fun and creative web application that allows users to express themselves through drawing and painting. It uses modern web development technologies such as HTML, CSS, and JavaScript to provide a user-friendly interface and a variety of color and brush options.
